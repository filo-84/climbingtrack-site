Kontolöschung (Deutsch)

Nutzer können ihr Konto direkt in der ClimbingTrack-App dauerhaft löschen:

Einstellungen → Konto löschen

Alternative Löschanfrage

Falls kein Zugriff auf die App mehr besteht, kann eine Löschanfrage per E-Mail gestellt werden:

support@deinemail.de

Gelöschte Daten

  • Accountdaten (E-Mail-Adresse, Nutzer-ID)
  • Gespeicherte Begehungen
  • Bewertungen und Kommentare

Aufbewahrung

Die Daten werden aus der aktiven Datenbank gelöscht. Backups können aus technischen Gründen bis zu 30 Tage bestehen bleiben.

Delete Account (English)

Users can permanently delete their account directly in the ClimbingTrack app:

Settings → Delete Account

Alternative request

If you cannot access the app anymore, you can request deletion by email:

support@deinemail.de

Deleted data

  • Account data (email address, user ID)
  • Saved climbing ticks
  • Ratings and comments

Retention

Data is removed from the active database. Backups may remain for up to 30 days for technical reasons.
