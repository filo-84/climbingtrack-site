Kontolöschung

Nutzer können ihr Konto direkt in der ClimbingTrack-App dauerhaft löschen:

Menü -> Profil -> Benutzer löschen

Alternative Löschanfrage

Falls kein Zugriff auf die App mehr besteht, kann eine Löschanfrage per E-Mail gestellt werden:

climbingtrack@gmail.com

Hinweis

Personenbezogene Daten werden bei Kontolöschung auf dem Server sofort gelöscht und nicht mehr herausgegeben. Begehungen, Bewertungen und Kommentare werden anonymisiert. Inhalte, die bereits auf anderen synchronisierten Geräten gespeichert wurden, können dort bis zur nächsten Synchronisation vorübergehend lokal vorhanden sein. Weitere Informationen dazu finden Sie in unserer Datenschutzerklärung.

Delete Account

Users can permanently delete their account directly in the ClimbingTrack app:

Menu -> Profile -> Delete user

Alternative request

If you cannot access the app anymore, you can request deletion by email:

climbingtrack@gmail.com

Note

Personal data is deleted from the server immediately when an account is deleted and is no longer disclosed. Climbing ticks, ratings, and comments are anonymized. Content already stored on other synchronized devices may temporarily remain there locally until the next synchronization. For more details, see our Privacy Policy.
